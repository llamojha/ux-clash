- **CSS sanitization** — new `sanitizeCss()` strips `@import` and external `url()` references
- **Race condition fix** — removed SELECT+INSERT pattern, now relies on unique index + Postgres error code `23505` for clean 409
- **Dynamic OG image** — `opengraph-image.tsx` using Next.js `ImageResponse` for real PNG output (SVG unsupported by most platforms)
- **Logo colors** — red (`#ef4444`) left side, green (`#22c55e`) right side, competitive "versus" vibe
- **Full Spanish (es_ES)** — all UI strings, toasts, button labels, not-found pages, and DB seed data translated
- **API errors stay English** — machine-consumed JSON, not user-facing
- **"Top Entries" / "Leaderboard"** — kept as product terms (common in Spanish gaming contexts)
- **`supabase/` stays in `.gitignore`** — not committing migration files to repo
- **Removed `next-themes`** — unused dependency (dark theme hardcoded)
