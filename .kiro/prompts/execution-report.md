# Execution Report

Review and analyze the implementation you just completed.

## Generate Report

Save to: `.agents/execution-reports/[feature-name].md`

### Include:
- **Meta Information**: Plan file, files added/modified, lines changed
- **Validation Results**: Linting, type checking, tests (pass/fail)
- **What Went Well**: Specific things that worked smoothly
- **Challenges Encountered**: Difficulties and why
- **Divergences from Plan**: What changed, why, and whether justified
- **Skipped Items**: What was not implemented and why
- **Recommendations**: Process improvements for next time
