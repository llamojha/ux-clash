**Source:** https://github.com/midudev/hackaton-cubepath-2026

- Start: 18 March 2026
- Deadline: 31 March 2026, 23:59:59 CET
- Final voting (live): 1 April 2026, 18:00 CET

## Rules
1. **Hosted on CubePath** — App must be deployed on CubePath. Register at midu.link/cubepath (15$ free credit).
2. **New project** — Can be pre-existing code but must NOT have had real users or be an already-running product.
3. **Public repo + live demo** — Repository must be public with an accessible link to the demo on CubePath.
4. **Documentation** — Must include a `README.md` with: description, demo link, screenshots/GIFs, and explanation of how CubePath was used.
5. **Register via Issue** — Create an issue in the hackathon repo using the template.
6. **Teams** — If team project, only one person creates the issue. Prize goes to issue creator.
7. **Appropriate content** — No offensive, discriminatory, or inappropriate content.
8. **Functional project** — Must be working at time of review.

## Evaluation Criteria (by importance)
1. 🎨 **User experience** (UX)
2. 💡 **Creativity**
3. 🔧 **Project utility**
4. ⚙️ **Technical implementation**
